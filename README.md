# DDDnyang
반려동물 사진업로드, 공유 홈페이지

## 사용스택
- Java 11
- Spring framework
- Tomcat9
- Mybatis
- MySQL

## 기능
*  공통
    + 메인화면
    + 계정관리 : 로그인, 회원가입
    + 게시물 : 댓글, 북마크, 신고기능
    + 공지사항 : 공지사항
    + 마이페이지 : 사용자 작성글, 북마크한 글 조회 

* 관리자
    + 사용자관리 : 게시물 관리, 회원 관리, 신고글 관리, 검색기능
